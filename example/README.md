# How to use

1. Install dependencies:

   ```bash
   yarn install
   cd example
   yarn install
   ```

2. Install iOS pods:

   ```bash
   yarn pod:install
   ```

3. Install iOS dev build (only needed once or after rebuilding native dependencies):

   ```bash
   yarn ios:installDevBuild
   ```

4. Start the example app on iOS simulator:

   ```bash
   yarn ios:start
   ```

# Dark Theme Support

To switch theme press `Cmd + Shift + A` in iOS simulator.
